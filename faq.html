🍮 晶凍健康食品 / 產品

晶凍是什麼？

+

晶凍是一種果凍狀的保健食品劑型，質地軟滑易食用，適合添加各類營養素，特別適合長輩、兒童與吞嚥能力較弱的族群。

愷樂生醫有哪些晶凍產品？

+

目前自有三款晶凍：GABA 鈣鎂晶凍、左旋麩醯胺酸晶凍，以及主打鳳梨酵素與益生菌的樂暢氏晶凍；另結合福爾 FORA 晶凍、膠原蛋白與牙科保健系列，其他晶凍新品開發中。

樂暢氏晶凍是什麼？

+

樂暢氏晶凍是愷樂生醫含鳳梨酵素與益生菌的晶凍健康食品，主打日常消化道保健的飲食補充。屬一般食品，相關成分研究累積中、台灣未核准功效宣稱，不具醫療效能；目前開發中、即將推出。

晶凍有哪些口味與型式？

+

愷樂自有晶凍為條狀隨身包、軟滑好入口、免配水，採純素配方、幾乎無糖，適合長輩、孩童與想長期、輕鬆補充營養的人。

GABA 鈣鎂晶凍適合誰？怎麼吃？

+

GABA 鈣鎂晶凍含鈣、鎂與 GABA，適合女性、銀髮族與外食族補充鈣鎂；建議每日 1 包、直接食用免配水。依台灣公告營養素功能宣稱，鈣有助於骨骼與牙齒正常發育、鎂有助於正常生理代謝。為一般食品，不具醫療效能。

左旋麩醯胺酸晶凍適合誰？怎麼吃？

+

左旋麩醯胺酸晶凍適合上班族、外食族與運動族群的日常與運動後營養補給；建議每日 1～3 包、直接食用免配水。屬一般食品，相關成分研究累積中、台灣未核准功效宣稱。

愷樂晶凍是純素的嗎？有什麼特色？

+

自有晶凍採純素配方、幾乎無糖（每包約 2.5 大卡），台灣製造、每批把關重金屬與微生物檢驗，成分與含量清楚標示。

愷樂晶凍如何保存？可以放多久？

+

建議存放於陰涼乾燥處、避免陽光直射與高溫；開封後盡快食用，並在外包裝標示的有效期限內食用完畢。

晶凍可以每天吃嗎？孕婦或兒童適合嗎？

+

晶凍為一般食品，可作為日常營養補充，建議依包裝建議量食用。孕婦、授乳者、服藥者或特殊疾病者，食用前建議先諮詢醫師或藥師。

IM8 適合哪些人？

+

IM8 適合忙碌的現代人、外食族、銀髮族、運動族群與想以簡單方式補足每日營養的人群。

IM8 每天怎麼吃、多久有感？

+

每日建議補充一包，搭配水或常溫飲品沖泡飲用。因個人體質與作息不同而有差異，多數人建議規律補充 4 至 8 週並維持均衡生活，感受較為穩定。

🩺 FORA 醫療器材

血糖機如何挑選？

+

可從準確度（ISO 15197）、採血量、試紙成本、操作便利性與售後服務五個面向評估。詳見血糖機完整選購指南。

血糖機準確度怎麼看？

+

主要參考國際標準 ISO 15197，確認測量值落在實驗室參考值的合理誤差範圍內。可定期使用校正液確認準確度。

居家血糖機測值和醫院抽血不同，正常嗎？

+

略有差異屬正常現象。居家血糖機多為微血管全血量測，醫院為靜脈血漿檢驗，兩者方法不同；只要落在合理誤差範圍內即可作為日常追蹤參考。

血糖試紙如何保存？

+

存放於陰涼乾燥處、避免陽光直射與潮濕，每次取用後立即蓋緊瓶蓋，並在有效期限內使用完畢。

血糖試紙可以不同機型混用嗎？

+

不建議。試紙需搭配相對應機型與校正碼使用，混用可能導致數值不準確，請使用原廠對應的試紙。

血壓計選臂式還是腕式？

+

一般建議臂式血壓計，量測位置接近心臟、數據較穩定準確；腕式輕巧適合外出但較易受姿勢影響。詳見血壓計完整選購指南。

血壓什麼時候量最準確？

+

建議在固定時間量測，例如晨起與睡前；量測前先靜坐休息約 5 分鐘，避免咖啡因、運動或情緒激動後立即測量，並保持手臂與心臟同高。

額溫槍與耳溫槍有什麼差別？

+

額溫槍量測額頭表面溫度、非接觸快速適合大量篩檢；耳溫槍量測耳膜紅外線溫度、較接近核心體溫。兩者量測部位不同，判讀時建議參考各自的正常範圍。

體溫幾度算發燒？

+

一般以耳溫或肛溫達 38°C 以上視為發燒，但會因量測部位與個人基礎體溫而略有差異。若持續高燒或合併不適，建議就醫評估。

FORA 醫療器材有提供保固嗎？

+

FORA 福爾醫療器材主機提供原廠保固，實際保固期間與範圍依產品型號而定，購買時可向業務團隊確認。

💡 成分・選購・法規

GABA 是什麼？

+

GABA（γ-胺基丁酸）是人體可自行合成的胺基酸，也存在於發芽米、番茄等食物，常見於夜間營養相關的多元配方。相關研究累積中、台灣未核准功效宣稱。詳見成分知識中心。

左旋麩醯胺酸是什麼？

+

左旋麩醯胺酸（L-Glutamine）是人體含量最高的游離胺基酸之一，常見於運動營養與保健配方。相關研究累積中、台灣未核准功效宣稱。詳見成分知識中心。

鈣和鎂為什麼常一起補充？

+

鈣與鎂是兩種重要礦物質，常被一起討論與搭配。依台灣公告營養素功能宣稱，鈣有助於骨骼與牙齒正常發育、鎂有助於正常生理代謝與神經肌肉功能。詳見鈣 vs 鎂比較。

晶凍和膠囊、錠劑比有什麼優點？

+

晶凍可撕開直接食用、免配水、軟滑好入口、單條好攜帶，對長輩與孩童的接受度通常高於膠囊或錠劑，較容易養成每日補充的習慣。詳見劑型比較中心。

鳳梨酵素與益生菌是什麼？

+

鳳梨酵素是萃取自鳳梨的蛋白質分解酵素；益生菌是對人體有益的活性菌，兩者與日常消化道保健常被一起討論。相關健康宣稱受法規限制，作為一般食品不宣稱療效。

福爾 FORA 晶凍、膠原蛋白與牙科系列是什麼？

+

除自有晶凍外，愷樂生醫結合福爾 FORA 晶凍、膠原蛋白與牙科保健系列（如牙科膠原蛋白、再生膜等），擴充醫美保養與口腔牙科照護的產品線，陸續推出。
